Döviz Kuru Uygulaması

Bu proje, Dolar (USD), Euro (EUR) ve Türk Lirası (TRY) arasında anlık kur dönüşümü yapan basit bir web uygulamasıdır.

Özellikler

* Dolar, Euro ve Türk Lirası arasında çift yönlü çeviri.
* Anlık ve güncel kur verileri.
* Sade ve kullanıcı dostu arayüz.

Kullanılan Teknolojiler

* **React:** Kullanıcı arayüzü için.
* **Vite:** Geliştirme ortamı için.
* **CSS:** Projenin stil ve tasarımı için.

Projeyi Bilgisayarda Çalıştırma

1.  **Projeyi klonlayın:**
    ```bash
    git clone [https://github.com/beratsrsln/doviz-kuru-uygulamasi.git](https://github.com/beratsrsln/doviz-kuru-uygulamasi.git)
    ```

2.  **Proje klasörüne gidin:**
    ```bash
    cd doviz-kuru-uygulamasi
    ```

3.  **Gerekli paketleri yükleyin:**
    ```bash
    npm install
    ```

4.  **Projeyi başlatın:**
    ```bash
    npm run dev
    ```
